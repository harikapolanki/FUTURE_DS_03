# Marketing Funnel & Conversion Analysis

This project analyzes the customer journey from Visitors → Leads → Customers.

## Tools Used
- Microsoft Excel

## Analysis Performed
- Visitor-to-Lead Conversion Rate
- Lead-to-Customer Conversion Rate
- Funnel Drop-off Analysis

## Key Insights
- LinkedIn and Email showed the highest conversion rates.
- Facebook had the lowest Visitor-to-Lead conversion.
- Most users dropped off before becoming leads.

## Recommendations
- Improve Facebook targeting.
- Increase focus on LinkedIn and Email campaigns.
- Optimize landing pages.
